# Introduction

Bienvenue dans la documentation de **ohada-lib** !

## Qu'est-ce que ohada-lib ?

**ohada-lib** est une bibliothèque TypeScript conçue pour simplifier la gestion comptable selon les normes **SYSCOHADA**. Elle permet aux développeurs de transformer des événements métier (ventes, achats, charges) en écritures comptables conformes, prêtes pour l'audit.

### Pourquoi utiliser ohada-lib ?

- **Conformité native** : Plus besoin de mémoriser les codes de comptes complexes (4111, 701, etc.). La bibliothèque les résout automatiquement.
- **Type-Safe** : Détectez les erreurs de logique comptable dès la compilation grâce à la puissance de TypeScript.
- **Zéro dépendance** : Légère et rapide, elle s'intègre partout.
- **Support Multi-pays** : Conçue pour s'adapter aux spécificités des différents pays de la zone OHADA.
