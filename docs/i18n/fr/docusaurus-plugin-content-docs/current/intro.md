# Introduction

Bienvenue dans **ohada-lib**, le moteur TypeScript professionnel pour une gestion financière conforme dans l'espace OHADA.

## Pourquoi ohada-lib ?

La comptabilité pour la région OHADA (Organisation pour l'Harmonisation en Afrique du Droit des Affaires) implique des réglementations strictes, un plan comptable spécifique (SYSCOHADA) et des règles méticuleuses de double étape pour les transactions à crédit.

Les logiciels de comptabilité traditionnels sont souvent trop lourds ou déconnectés du flux de travail des développeurs. **ohada-lib** a été conçu avec une mission simple :

> **"La comptabilité OHADA doit être simple pour tout développeur."**

## Piliers Fondamentaux

- **🚀 Performance** : Un moteur sans dépendance pesant seulement ~70Ko. Déployez-le sur des fonctions edge, des applications mobiles ou des serveurs centraux.
- **📐 Conformité** : Logique intégrée pour le standard SYSCOHADA révisé. Chaque écriture est pré-validée selon les règles officielles.
- **🧠 Résolution Intelligente** : Mappe automatiquement les événements métiers vers les bons codes de compte. Arrêtez de chercher manuellement le 4111 ou le 701.
- **🛡️ Type-Safety** : Construit avec TypeScript pour capturer les erreurs logiques dès la compilation, réduisant les risques d'audit.
- **🌍 Multi-Langue** : Support natif du français et de l'anglais, couvrant le paysage commercial diversifié des 17 États membres.
