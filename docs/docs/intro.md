# Introduction

Welcome to the **Ohada Lib** documentation. 

Ohada Lib is a developer-friendly OHADA accounting library designed to transform business events into type-safe journal entries, manage stock, and expenses, and generate compliant financial reports for the OHADA region.

## Key Features

- **Direct Accounting**: Use simple verbs like `recordPurchase` or `recordSale`.
- **OHADA Compliant**: Automatically generate journal entries following SYSCOHADA standards.
- **Internationalization**: Support for French and English locales.
- **Self-Resolving Accounts**: Smart identification of accounting codes based on item labels.
- **Type-Safe**: Built with TypeScript for a robust developer experience.

## Why Ohada Lib?

Traditional accounting systems are often rigid and complex. Ohada Lib bridge the gap between business operations and accounting compliance, allowing developers to focus on building great features while the library handles the underlying accounting logic.
