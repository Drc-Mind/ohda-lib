# Internationalization (i18n)

Ohada Lib supports multiple locales for journal entry labels. This ensures that your accounting reports are generated in the language your users expect.

## Configuration

Set the `locale` property when initializing the `Ohada` class.

```typescript
const ohada = new Ohada({
  locale: 'en' // Options are 'fr' (default) or 'en'
});
```

## Localized Labels

When a locale is set, the following labels are automatically translated:
- Purchase (Generic) / Achat (Générique)
- Supplier / Fournisseur
- Cash Outflow / Sortie de trésorerie
- Recoverable VAT / TVA récupérable
- etc.

## Example

```typescript
const ohadaEn = new Ohada({ locale: 'en' });
const res = ohadaEn.recordPurchase(5000);

// Journal entries will contain English labels
console.log(res.journalEntries[0].label); // "Purchase (Generic)"
```
